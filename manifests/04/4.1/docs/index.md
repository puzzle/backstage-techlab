# My Sample Service


## Overview

This is a sample microservice that demonstrates Backstage catalog integration.


## Architecture

The service is built with Node.js and provides a REST API for user management.


## Getting Started


### Prerequisites

* Node.js 22+
* PostgreSQL 14+


### Installation

```bash
npm install
npm start
```
